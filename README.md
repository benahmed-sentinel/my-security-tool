
# 🔥 SentinelRecon

A powerful Python-based reconnaissance and security scanning tool.

---

## ⚡ Features

- 🔍 WHOIS Lookup
- 🌐 DNS Enumeration
- 📡 Traceroute
- 🔓 Port Scanning
- 🛠 Service Detection
- 🌍 Web Server Detection
- 📶 UDP Scan
- ⚠️ Vulnerability Scan (Nmap Scripts)

---

## 🧰 Requirements

- Python 3.x
- Linux (recommended)

---

## 🚀 Usage

Run the tool using:

```bash
python security_tool.py example.com
```
