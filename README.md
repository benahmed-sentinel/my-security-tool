# SentinelRecon 🔥

أداة فحص أمني واستطلاع (Reconnaissance Tool) مطورة بلغة Python، مخصصة لجمع المعلومات وتحليل الأهداف بشكل منظم.

## ⚡ المميزات

- 🔍 WHOIS Lookup
- 🌐 DNS Enumeration
- 🛣️ Traceroute
- 🔓 Port Scanning
- 🔧 Service Detection
- 🌍 Web Server Detection
- 📡 UDP Scan
- ⚠️ Vulnerability Scan (Nmap Scripts)
- 🔐 SSL/TLS Analysis
- 📍 IP Information
